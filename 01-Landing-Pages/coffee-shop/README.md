# Espresso House

Espresso House adalah template landing page coffee shop premium yang dibangun dengan HTML, Tailwind CSS, dan asset statis.

## Fitur
- Landing page modern dan responsif
- Halaman About, Menu, Reservasi, Review, Gallery, dan 404
- Favicon brand yang konsisten
- Gambar yang dioptimalkan dalam format WebP

## Cara Instalasi
1. Clone atau unduh project ini.
2. Buka folder project di browser, atau jalankan server lokal sederhana.
3. Jika ingin menjalankan lewat server lokal, gunakan perintah berikut:

```bash
python -m http.server 8000
```

4. Buka http://localhost:8000 di browser.

## Struktur Proyek
- index.html: halaman utama
- about.html: halaman tentang kami
- menu.html: halaman menu
- reservasi.html: halaman reservasi
- Review.html: halaman review pelanggan
- gallery.html: halaman gallery
- 404.html: halaman not found
- assets/css/: stylesheet
- assets/js/: file JavaScript interaktif
- assets/images/: gambar utama dalam format WebP
- assets/icons/: favicon dan ikon pendukung

## Catatan
Project ini bersifat statis, sehingga tidak membutuhkan build process atau instalasi dependency tambahan.
